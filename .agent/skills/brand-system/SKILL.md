## 🛠️ Implementation Checklist

Before delivering code, verify it adheres to the PECAÊ Identity:

- [ ] **Glassmorphism:** Backdrop blur is set to `16px`?
- [ ] **No-Line Rule:** Are you using tonal shifts instead of solid 1px borders?
- [ ] **Typography:** Are you using `Space Grotesk` for technical data and `Manrope/Inter` for body?
- [ ] **Glow vs Shadow:** Are you using **Ambient Glows** in Dark Mode instead of black shadows?
- [ ] **Light Mode:** Is the glass "suave e limpo" (soft white/mint gradients)?
- [ ] **Touch Targets:** Are primary buttons and nav items in the Thumb Zone?

---

## ⚠️ CRITICAL: The "Digital Forge" Rule

**Never use common marketplace templates.** 
- PECAÊ should feel like a high-end diagnostic interface.
- Use intentional asymmetry.
- Overlap images on glass surfaces to create 3D depth.
