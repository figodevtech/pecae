# Roadmap PECAÊ — Do MVP à Escala

Este roadmap define as fases de construção do marketplace Peçaê, priorizando o fluxo de valor para vendedores e compradores.

## 🏁 Fase 1: Fundação e MVP (Atual)
*Foco: Registro, Cadastro de Sucatas e Busca Básica*

- [x] **M01: Gestão de Contas**
  - [x] Login/Cadastro Base
  - [x] Verificação de E-mail (Resend/OTP)
  - [x] Login Social (Google/Apple) - (Lógica base pronta no Controller)
- [x] **M03: Perfil do Vendedor**
  - [x] Cadastro de Endereço/Geolocalização
  - [x] Upload de Documentos para Verificação (Mocked Storage)
### M05: Gestão de Sucatas (Vendedor)
- **Status**: ✅ Concluído
- **Progresso**: 100%
- **Detalhes**: Fluxo de cadastro de sucatas, inventário de peças e upload de fotos implementado e sincronizado com a API.
### M07: Busca e Catálogo (Comprador)
- **Status**: ✅ Concluído
- **Progresso**: 100%
- **Detalhes**: Busca Full-Text robusta e filtros avançados por marca, modelo e ano. Integração mobile completa com novo componente de filtros.

### Chat Base
- **Status**: ✅ Concluído
- **Progresso**: 100%
- **Detalhes**: Canais de chat por anúncio funcionais. Notificações Push e In-app integradas ao envio de mensagens.

## 📊 Métricas de Sucesso
- **Ativação**: Cadastro do primeiro veículo pelo vendedor.
- **Retenção**: Comprador volta para fazer uma nova busca em < 30 dias.
- **Conversão**: Abertura de chat por anúncio visualizado.
