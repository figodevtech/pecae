# Arquivo de Histórico — REQUIREMENTS Milestone 2 (v2.0)

Este documento registra os Requisitos Funcionais, Regras de Negócio e Casos de Uso que foram 100% satisfeitos e validados ao término da **Milestone 2** do PECAÊ.

---

## Requisitos Funcionais Satisfeitos

### M02 — Perfil do Comprador
- [x] **RF10:** Comprador deve poder visualizar seu perfil com nome, e-mail verificado e telefone (P0)
- [x] **RF11:** Comprador deve poder editar nome e telefone (P0)
- [x] **RF12:** Comprador deve poder fazer upload de avatar (P2)
- [x] **RF13:** Comprador deve ter acesso centralizado a favoritos, buscas salvas e conversas (P0)
- [x] **RF14:** Comprador deve poder configurar preferências de notificação por canal (push, e-mail, in-app) (P1)
- [x] **RF15:** Comprador deve poder excluir conta e dados pessoais (LGPD) (P1)

### M03 — Perfil do Vendedor
- [x] **RF30:** Vendedor deve poder criar perfil com nome da loja, tipo (PF/PJ), telefone, WhatsApp, endereço, cidade e estado (P0)
- [x] **RF31:** Vendedor deve poder fazer upload de logo da loja (P1)
- [x] **RF32:** Vendedor deve poder editar seu perfil a qualquer momento (P0)
- [x] **RF33:** Sistema deve exibir página pública do vendedor com seus anúncios ativos (P0)
- [x] **RF34:** Vendedor deve poder solicitar o Selo Verificado enviando documentação (P1)
- [x] **RF35:** Sistema deve calcular e exibir estatísticas do vendedor (tempo de resposta, anúncios ativos, total de vendas) (P1)
- [x] **RF36:** Comprador deve poder visualizar o perfil público de qualquer vendedor (P0)

## Regras de Negócio Implementadas

### M02 — Perfil do Comprador
- [x] **RN07:** Comprador responsável pela veracidade de seus dados pessoais.
- [x] **RN-M02-01:** E-mail do comprador só pode ser alterado mediante reverificação via link enviado ao novo e-mail.
- [x] **RN-M02-02:** Exclusão de conta é soft delete síncrono com oclusão total de anúncios e veículos imediata; dados pessoais anonimizados após 30 dias (LGPD).
- [x] **RN-M02-03:** Comprador pode ter múltiplas buscas salvas, mas máximo 10 ativas simultaneamente.

### M03 — Perfil do Vendedor
- [x] **RN13:** Selo de Verificado é concedido exclusivamente pela moderação após validação documental.
- [x] **RN07:** Vendedor é inteiramente responsável pela veracidade das informações publicadas no perfil.
- [x] **RN08:** Placa de veículos é exibida mascarada; CNPJ do vendedor PJ é exibido apenas parcialmente no perfil público.
- [x] **RN-M03-01:** Um usuário pode ter apenas um SellerProfile ativo.
- [x] **RN-M03-02:** Vendedor com status SUSPENDED ou BANNED (ou DELETED) tem perfil oculto dos resultados públicos.
- [x] **RN-M03-03:** SellerStats são calculados assincronamente via jobs; não são calculados em tempo real.
